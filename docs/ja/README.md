## Edgion について

Edgion は Pingora と Kubernetes Gateway API をベースとした高性能ゲートウェイです。

**特徴：**
- ✅ Gateway API v1 サポート
- ✅ TCP/UDP ストリームプラグイン
- ✅ mTLS 相互認証
- ✅ 動的証明書ロード（SNI ベース）
- ✅ IP アクセス制御
- ✅ Pingora ベースの高性能プロキシ
